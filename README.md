Colorpicker
===========

A JQuery UI Colorpicker Widget.
* Supports RGB, CMYK, HSB and the [B&A Colorwheel](http://de.scribd.com/doc/119162621/Ba-Color-Wheel) (a predefined colowheel)
* Includes a generator for several color harmonies
